# Square Crop

画像を正方形にトリミングし、元画像とセットでZIPダウンロードできるシンプルなWebツール。

## 特徴

- **ブラウザ完結**: 全ての処理はローカルで実行。画像データはサーバーに送信されません
- **複数画像対応**: 一度に複数の画像を読み込み、効率よく処理
- **正方形トリミング**: ドラッグ＆リサイズで簡単に切り出し範囲を指定
- **一括ダウンロード**: トリミング画像と元画像をセットでZIP出力

## 使い方

1. 画像をドラッグ＆ドロップ（複数可）
2. マウスで正方形枠を配置・サイズ調整
3. ←→キーまたはボタンで画像を切り替え
4. 出力サイズを選択（512 / 1080 / 2048 / 3000 px またはカスタム）
5. 「Download ZIP」をクリック

## 出力ファイル

ZIPファイルには以下の形式で保存されます：

- `1-1.png` - 1枚目のトリミング画像
- `1-2.jpg` - 1枚目の元画像（元の拡張子を維持）
- `2-1.png` - 2枚目のトリミング画像
- `2-2.png` - 2枚目の元画像
- ...

## 操作方法

| 操作 | 説明 |
|------|------|
| ドラッグ＆ドロップ | 画像を読み込み |
| クリック | ファイル選択ダイアログを開く |
| 枠をドラッグ | トリミング位置を移動 |
| 四隅をドラッグ | トリミングサイズを変更（正方形固定） |
| → キー | 次の画像へ |
| ← キー | 前の画像へ |

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- JSZip（ZIP生成）
